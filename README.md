# Покупки домой

Приватный навык Алисы для Яндекс Диалогов: принимает голосовой ввод пользователя и отправляет список покупок в Telegram-чат.

## Что умеет

- Принимает `POST`-запросы от Yandex Dialogs на `/webhook`.
- На первой реплике новой сессии отвечает: `Говорите, что добавить в список покупок`.
- Следующие реплики отправляет в Telegram через Bot API.
- Завершает сессию по словам: `завершить`, `стоп`, `хватит`, `выход`.
- Хранит секреты только в переменных окружения.

В первой версии нет базы данных, авторизации, чтения списка, удаления товаров, категорий и нескольких списков.

## Переменные окружения

Создайте `.env` для локального запуска или задайте переменные в окружении деплоя:

```bash
TELEGRAM_BOT_TOKEN=123456789:replace_me
TELEGRAM_CHAT_ID=123456789
```

`TELEGRAM_BOT_TOKEN` можно получить у BotFather.

`TELEGRAM_CHAT_ID` - это ID чата, куда бот будет отправлять покупки. Бот должен быть добавлен в этот чат.

## Запуск локально

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN="123456789:replace_me"
export TELEGRAM_CHAT_ID="123456789"
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Если вы храните локальные секреты в `.env`, запускайте так:

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000 --env-file .env
```

Локальный webhook будет доступен по адресу:

```text
http://localhost:8000/webhook
```

Для Яндекс Диалогов нужен публичный HTTPS-адрес. Для локальной проверки можно временно пробросить порт через ngrok, cloudflared tunnel или другой HTTPS-туннель:

```bash
ngrok http 8000
```

После этого используйте HTTPS URL вида:

```text
https://your-domain.example/webhook
```

## Деплой

Подойдет любой хостинг, который умеет запускать Python ASGI-приложение и отдавать HTTPS: Render, Railway, Fly.io, VPS, Yandex Cloud и другие.

Общий порядок:

1. Загрузите проект на хостинг.
2. Установите зависимости из `requirements.txt`.
3. Задайте переменные окружения `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`.
4. Запустите приложение командой:

```bash
uvicorn main:app --host 0.0.0.0 --port $PORT
```

Если платформа не задает `$PORT`, используйте порт, который требует конкретный хостинг.

## Подключение webhook в Яндекс Диалогах

1. Откройте консоль Яндекс Диалогов.
2. Создайте приватный навык с названием `Покупки домой`.
3. В настройках backend выберите webhook.
4. Укажите публичный HTTPS-адрес:

```text
https://your-domain.example/webhook
```

5. Сохраните настройки и проверьте навык в тестовом чате.

## Логика диалога

Первая реплика новой сессии:

```text
Алиса: Говорите, что добавить в список покупок
```

Пользователь:

```text
Молоко, хлеб и яйца
```

Навык отправит `Молоко, хлеб и яйца` в Telegram и ответит:

```text
Добавила. Говорите еще или скажите завершить
```

Пользователь:

```text
завершить
```

Навык ответит:

```text
Хорошо, закрываю список
```
